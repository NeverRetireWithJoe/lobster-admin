【1. 推送到 GitHub】

  方式一（推薦）:
    cd /home/autorun/.openclaw/workspace/lobster-admin
    ./PUSH_TO_GITHUB.sh

  方式二（手動）:
    cd /home/autorun/.openclaw/workspace/lobster-admin
    
    # 先在 GitHub 創建 repository: lobster-admin
    
    git remote add origin https://github.com/YOUR_USERNAME/lobster-admin.git
    git branch -M main
    git push -u origin main

【2. 部署到 Vercel（前端）】

  A. 訪問 vercel.com 並登入
  B. 點擊 "New Project"
  C. Import 你的 GitHub repository
  D. 配置:
     - Framework: Vite
     - Root Directory: frontend
     - Build Command: npm run build
     - Output Directory: dist
  E. 環境變數:
     - VITE_API_URL = http://YOUR_VPS_IP:8080
  F. Deploy!

【3. 後端持久運行（VPS）】

  使用 PM2（推薦）:
    npm install -g pm2
    cd /home/autorun/.openclaw/workspace/lobster-admin
    pm2 start ecosystem.config.js
    pm2 save
    pm2 startup

  或使用內建腳本:
    ./run-background.sh start

【4. 安全設定】

  ⚠️ 登入後立即修改預設密碼
  ⚠️ 修改 backend/auth.js 中的 JWT_SECRET
  ⚠️ 設定 HTTPS（使用 Nginx + Let's Encrypt）
  ⚠️ 配置防火牆規則
